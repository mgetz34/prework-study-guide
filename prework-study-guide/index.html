HTML

  • The head element contains information about the webpage.
  • The body element contains the visible content shown to the user.
  • The head and body elements are bracketed by opening and closing tags.

CSS

  • CSS or cascading style sheets style the the HTML content.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element
  • The separation of concerns principal states that each section of code has it's own responsibility
  • Class attributes add style to certain elements but not all to break up the formating of a web document

Git

  • git status: checks what branch we are currently working on
  • git checkout -b branch name: creates a new branch and switches to it
  • make and save changes often with commits so the code doesent deviate too far from prior versions

JavaScript

  • A variable is a named comtainer that allows us to store data in our code.
  • Control flow is the order in which a computer executes in a script.
  • Interrupting the conrol flow allows developers to build applications that respond to user actions.
  • A function is a set of instructions that tells the computer to perform a certain task and when.
